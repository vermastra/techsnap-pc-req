PC Requirements
Author : Aditya Verma   Last Updated : 30 July, 2021

Beginners are required to have a laptop computer that meets the following minimum specifications,

For Programming :

Processor :  Intel core i5 or better processor, 7th generation or newer(Virtualization must be supported)
Operating System : Windows 10
Screen resolution : 1920*1080
Storage : minimum 500GB SSD is required, since SSD is faster than HDD
RAM : minumum of 8GB and must be Expandable
Webcam : required
graphic card : minimum 2GB integrated
Internet : high speed

For Gaming :

Processor :  Intel core i7-4770K procesor
Operating system : Windows 10(64-bit)
Screen resolution : 1920*1080
Storage : 2-3TB HDD, 1IB SSD
RAM : 16GB is best
Webcam : required
Graphic card : NVDIA GeForce GTX 1070
Internet : high speed

Windows 10 is basic requirement as some applications will not be supported by lower verions.
The above given resolution is not mandatory but it is better to detect tiny errors, inorder to make it more precise and to make our job easier. Whereas for gaming, it is highly recommended to have better gaming experience.
Programming and Gaming requires a lot of memory because our programs use up every last available megabyte while compiling. When we go for bigger projects, running programs may gets slower and also to implement the codes. Most probably the gaming experience gets interrupted. Therefore, we need to have larger RAM's.
You may wonder the need of webcam for programming right? Well, when we go for deep learning and Artificial intelligence technologies, absolutely we need a webcam to detect our face or things. Nowadays it is usual that having interviews, meetings perhaps everything online so, there comes usage of webcam.
A graphic card is an expansion card for your PC that is responsible for rendering images to the display. High-end GPU's are used for gaming, ray tracing, graphics production, and even mining cryptocurrency. Hence, having a good specified graphic card is higly functional for gaming purpose.
High speed internet is very important for today's way of living. Nevertheless, what the application we are using will definitely needs high speed internet. However programming may not require internet but for documentation and syntax reference, we need internet. High speed internet help with gaming in general, but they’re not the be-all, end-all. In fact, the routes that your game data takes from your PC to the game server and back are often more important than your connection speed.

NOTE

The above given are mere requirements for anybody to get started with programming and gaming. You can have requirements with higher specifications than mentioned above for better experience.
